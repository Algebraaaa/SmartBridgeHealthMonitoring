## 📋 项目简介

本项目立足于**国家基础设施安全**的重大需求，创新性地融合了测绘学、土木工程与人工智能的前沿技术，旨在为大型桥梁提供一套**低成本、高精度、易部署、智能化**的变形监测解决方案。

### 🎯 核心技术

| 技术 | 解决的问题 |
|------|-----------|
| **GNSS PPP技术** | 解决传统监测对基站的依赖痛点，实现厘米级动态定位 |
| **多源卡尔曼滤波** | 解决单一传感器的数据缺陷，融合GNSS、倾斜、应变数据 |
| **深度学习模型** | 实现智能化状态评估与趋势预测 |

### ✨ 项目特色

- 🎯 **PPP高精度定位** - 无需基站即可实现厘米级动态监测，突破空间限制
- 🔗 **多源数据融合** - GNSS + 倾斜 + 应变，消除单一传感器盲区
- ⚡ **实时智能预警** - 卡尔曼滤波动态评估，异常情况毫秒级响应
- 💰 **成本优势** - 单点监测成本仅为传统进口设备的 1/10

## 🚀 快速开始

### 在线预览

直接在浏览器中打开 `index.html` 文件即可查看项目演示。

### 本地运行

```bash
# 克隆项目
git clone <repository-url>

# 进入项目目录
cd Innovation

# 使用任意HTTP服务器运行（可选，直接打开html也可）
# 方式1: Python
python -m http.server 8080

# 方式2: Node.js
npx serve .

# 方式3: VS Code Live Server 插件
# 右键 index.html -> Open with Live Server
```

访问 `http://localhost:8080` 查看项目。

## 🔮 未来展望

### Phase 1: 数字孪生深度融合
将监测数据实时驱动高保真的桥梁FEM（有限元）模型，实现结构内力的实时反演。

### Phase 2: 众包监测模式
利用过往车辆的车载传感器（手机IMU、车载GPS）数据，构建"车-桥协同"的泛在感知网络。

### Phase 3: 标准化与产品化
完善硬件的工业设计，通过相关行业标准认证，推动成果从实验室走向工程市场。

## 💡 技术栈

### 前端展示
- **HTML5 / CSS3** - 页面结构与样式
- **Tailwind CSS** - 原子化CSS框架
- **Chart.js** - 数据可视化图表
- **Glassmorphism** - iOS风格毛玻璃设计

### 核心算法
- **PPP (Precise Point Positioning)** - 精密单点定位
- **EKF (Extended Kalman Filter)** - 扩展卡尔曼滤波
- **Deep Learning** - 深度学习预测模型

### 硬件设备
- GNSS多频接收机
- 高精度倾斜传感器
- 光纤/电阻应变计
